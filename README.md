#  Retail Sales SQL Analysis

##  Overview

This project focuses on analyzing a retail sales dataset using SQL queries.
The goal was to practice working with databases and extract meaningful insights using SQL instead of relying only on pandas.

---

##  Tools Used

* Python
* SQLite
* Pandas (for loading and displaying data)
* Matplotlib / Seaborn (for basic visualization)

---

##  What was done

The dataset was first loaded into an SQLite database, and then SQL queries were used to analyze the data.

### Key analysis performed:

* Total sales calculation
* Sales by product category
* Top transactions
* Gender-wise spending
* Age-based filtering
* Monthly sales trends
